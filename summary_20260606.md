## 1. 완료된 사용자 요청 (Accomplished Requests)

* **🔐 게스트 로그인 계정 도용 차단 및 중복 닉네임 검증 구현 [LATEST]**
  - 로그인 모달에서 `⚡ 일회용 게스트로 시작하기`를 시도할 때, 입력한 닉네임이 기존에 이미 가입된 회원 닉네임(예: `간장` 등)인 경우 게스트 로그인을 차단합니다.
  - 가입된 닉네임인 경우 **"이미 등록된 정식 회원 닉네임입니다. 비밀번호를 입력하여 로그인해 주세요."** 라는 안내 경고와 함께 로그인을 불허합니다.
  - 또한, 현재 온라인 접속 목록(`onlineUsers`)을 조회하여 이미 해당 닉네임으로 접속해 있는 사용자가 있는 경우 **"현재 접속 중인 사용자가 있습니다. 다른 닉네임을 입력해 주세요."** 경고를 반환하여 닉네임 겹침 및 세션 충돌 현상을 원천 방지했습니다.
* **💬 실시간 대화방 정렬 및 실시간 업데이트 연동 완료**
  - Supabase snake_case 반환 데이터와 프론트엔드의 camelCase 기대 키 간 불일치를 해결하기 위해 `formatDbRoom` 및 `formatDbMessage` 포맷터를 `chatService.js`에 탑재했습니다.
  - 대화 송수신이 감지되면 즉시 대화방 목록을 리로드 정렬(`loadChatRooms()`)하도록 변경하여, 3초 타이머 딜레이 없이 1초 안에 실시간으로 대화방 정렬이 최상단으로 슉 이동합니다.
* **🕒 대화방 목록 업데이트 시간 표시 고도화**
  - 오늘 온 메시지는 `오전/오후 10:24`, 어제 온 메시지는 `어제`, 7일 이내 메시지는 `토요일`과 같은 요일, 그 이전은 `6월 6일`과 같이 카카오톡처럼 친숙하게 보여줍니다.
* **🚀 내 글 끌어올리기 (끌올) 기능 추가**
  - 내가 쓴 거래글을 삭제하고 다시 올릴 필요 없이, 클릭 한 번으로 피드 목록의 맨 위로 보낼 수 있는 끌어올리기(Bump) 기능이 추가되었습니다.
* **🛠️ 내 글 수정 기능 추가**
  - 피드 카드 내 '수정' 버튼 및 '내 정보' 드로어 목록에 '수정' 버튼을 추가하여, 기존에 올린 글을 간편하게 수정할 수 있게 구현했습니다.
  - 수정 모달창 내에 연락처 수정 기입란은 물론, 줄 수 있는 카드와 받고 싶은 카드를 모달 내에서 즉시 넣고 뺄 수 있도록 미려한 스티커 선택 패널(미니 도감 편집기)을 전격 탑재했습니다.
* **💬 댓글 기능 추가 (아코디언 토글)**
  - 교환소 피드 카드 하단에 댓글 펼치기 버튼을 제공하고, 클릭 시 해당 글에 대한 전체 댓글 목록과 함께 실시간 댓글 등록 폼을 지원합니다.
* **🚨 교환글 및 댓글 신고 기능 추가**
  - 타인이 등록한 교환글 또는 댓글에 대해 부적절한 사유로 신고할 수 있는 신고 접수 팝업창을 구현했습니다.
* **👑 관리자 전용 탭/대시보드 구현 (간장 계정 전용)**
  - 로그인 닉네임이 `간장` 일 경우, 헤더 닉네임 옆에 `👑 관리자` 골드 배지가 추가되어 관리자 대시보드로 원클릭 진입할 수 있습니다.
  - 관리자 대시보드에서는 접수된 실시간 신고 목록을 한눈에 모니터링하고, 허위 신고일 경우 '신고 반려' 처리하거나 유해 콘텐츠일 경우 '대상 강제 삭제 & 종결' 처리를 원버튼으로 강제 집행할 수 있습니다.
* **🗃️ Supabase 테이블 DDL 쿼리 스키마 제공**
  - 새로 구현한 댓글(`post_comments`) 및 신고(`reports`) 기능의 DB 테이블 생성을 원 클릭으로 돕기 위해 프로젝트 루트에 `supabase_schema.sql` 스키마 쿼리 파일을 신설했습니다.
