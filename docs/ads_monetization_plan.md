# 광고 수익화 적용 계획

## 1. 권장 경로

이 사이트는 커뮤니티성 도구이므로 Google AdSense를 우선 검토한다.

1. 독립 도메인 연결
2. 개인정보처리방침, 문의/운영자 안내, 커뮤니티 이용 규칙 페이지 준비
3. 저작권 문제가 있는 이미지/브랜드 사용 범위 점검
4. AdSense 신청
5. 승인 후 발급받은 publisher ID로 광고 스크립트 적용

## 2. 배치 원칙

- 모바일에서는 화면 하단 고정 채팅 버튼과 겹치지 않게 본문 사이 배너만 사용한다.
- 교환글 카드 내부에는 광고를 넣지 않는다.
- 게시판 상단 또는 교환글 목록과 게시판 사이에 반응형 배너 1개를 우선 배치한다.

## 3. 구현 메모

- 실제 광고 코드는 `ca-pub-...` publisher ID가 발급된 뒤 추가한다.
- 승인 전에는 광고 placeholder만 둘 수 있으며, 가짜 클릭 유도 문구는 넣지 않는다.
- CSP 적용 시 `pagead2.googlesyndication.com`, `googleads.g.doubleclick.net` 등 AdSense 도메인을 허용 목록에 추가해야 한다.

## 4. 운영 주의

- 게임 닉네임, 교환 코드, 채팅 내용처럼 사용자가 올린 정보는 광고 타겟팅용으로 별도 전송하지 않는다.
- 성인/도박/현금거래 표현이 게시판에 쌓이면 광고 심사와 계정 상태에 악영향이 있을 수 있어 관리 정책이 필요하다.
