# 📋 구글 애드센스 재신청 전 최종 체크리스트

이 문서는 드빌3 스티커교환소(`https://dv3-sticker.vercel.app/`)의 애드센스 재심사 통과를 위해, 신청 전에 운영자가 최종적으로 자가 진단해야 할 4가지 핵심 체크 포인트를 담고 있습니다.

---

## 1. Vercel 배포 상태 및 접속 점검 [ ]
* [ ] **배포 빌드 성공**: `feature/adsense-renewal` 브랜치를 Vercel에 연동한 후, 배포 빌드가 오류 없이 통과하는지 확인합니다.
* [ ] **도메인 연결**: `https://dv3-sticker.vercel.app/` 주소로 정상 접속이 되며, SSL 보안 인증(https)이 정상 활성화되었는지 확인합니다.
* [ ] **모바일 화면 최적화**: 스마트폰으로 접속했을 때 메인 안내 배너나 스티커 목록이 찌그러지거나 겹치지 않고 정상 노출되는지 확인합니다.

---

## 2. 404 깨진 링크 및 사이트맵(Sitemap) 점검 [ ]
* [ ] **백과사전 링크 확인**: 메인 페이지 하단 및 가이드 페이지 내의 `스티커 백과사전` 링크를 클릭했을 때, `/about-stickers.html` 공략 페이지가 404 에러 없이 200 OK로 잘 열리는지 확인합니다.
* [ ] **개인정보처리방침 공식 링크**: `/privacy.html` 내의 구글 파트너 사이트 데이터 사용 고지 링크(`https://policies.google.com/technologies/partner-sites`)가 새 창으로 정상 연결되는지 점검합니다.
* [ ] **sitemap.xml 검색 반영**: 주소창에 `https://dv3-sticker.vercel.app/sitemap.xml`을 입력해보고, 새로 추가된 `/about-stickers.html` 주소가 XML 내용 안에 정확히 들어가 있는지 최종 확인합니다.

---

## 3. 로그인 게이트 및 UGC 금칙어 테스트 [ ]
* [ ] **크롤러 우회 보장**: 쿠키나 로컬스토리지 데이터를 모두 지운 비로그인 브라우저(시크릿 창)로 접속했을 때, 로그인 화면이 자동으로 화면을 덮지 않고 메인 피드가 완전히 열려 있는지 확인합니다.
* [ ] **금칙어 차단 장치 검증**: 
  - 비로그인 상태로 게스트 로그인을 진행합니다.
  - 교환 등록 폼이나 자유게시판 글쓰기에 제목 또는 내용으로 **"현금 거래"**, **"계정 판매"**, **"문상 팝니다"** 등을 적어봅니다.
  - "커뮤니티 이용규칙 준수를 위해 금전 거래 유도 단어 및 금칙어는 포함할 수 없습니다"라는 경고 경고창이 뜨며 등록이 정상적으로 차단되는지 확인합니다.
* [ ] **로그인 모달 X 단추**: 닫기 버튼(`X`)을 눌렀을 때 팝업창이 자연스럽게 없어지는지 테스트합니다.

---

## 4. 구글 서치 콘솔 & 애드센스 최종 신청 [ ]
* [ ] **Google Search Console 색인 등록**: 
  - 구글 서치 콘솔에 접속하여 새로 생긴 주소인 `https://dv3-sticker.vercel.app/about-stickers.html` 에 대한 **색인 생성 요청**을 보냅니다.
* [ ] **구글 애드센스 콘솔 재검토 요청**:
  - 준비가 모두 완료되면, 구글 애드센스 웹사이트에 접속하여 **"수정 완료"**를 누르고 재심사를 신청합니다.
  - 심사는 통상적으로 3일에서 길게는 2주일 정도 소요됩니다.
