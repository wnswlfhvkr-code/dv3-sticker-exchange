연락부터 신고까지, 안전 교환 체크리스트

이 사이트는 교환 후보를 찾고 대화를 시작하도록 돕습니다. 실제 스티커 보유와 최종 교환 조건은 당사자가 게임에서 다시 확인해야 합니다.

1. 사이트가 확인하는 것과 확인하지 못하는 것

사이트에서 확인 가능

  • 사용자가 게시한 Haves와 Wants
  • 두 목록의 완전·부분 매칭 여부
  • 게시글에 적은 닉네임·연락처·설명
  • 사이트 안에서 접수된 신고 내용

사이트에서 확인 불가

  • 현재 게임 인벤토리와 실제 수량
  • 상대방의 신원 또는 신뢰도
  • 외부 링크와 메신저 계정의 소유자
  • 게임 안에서 최종 교환이 완료됐는지

매칭 표시가 있어도 실제 보유를 인증한 것은 아닙니다. 이 차이를 알고 확인 절차를 거치는 것이 가장 중요합니다.

2. 상대에게 연락하기 전

  • 일치한 스티커의 묶음 이름, 슬롯, 표시 이름을 메모합니다.
  • 내 Haves가 아직 실제 여분인지 게임에서 확인합니다.
  • 상대 글의 작성 시점과 목록이 최근 상태인지 질문합니다.
  • 연락처 링크의 도메인과 계정 이름을 열기 전에 확인합니다.
  • 비밀번호, 인증번호, 복구 코드, 결제 정보는 누구에게도 전달하지 않습니다.

3. 대화에서 합의할 내용

“야생 몬스터 하나”처럼 범위가 넓은 표현보다 스티커 이름과 슬롯을 함께 적는 편이 착오를 줄입니다.

확인 메시지 예시

제가 드릴 항목은 야생 몬스터 2번 슬롯(1-2), 받고 싶은 항목은 물속 몬스터 4번 슬롯(2-4)입니다. 두 항목을 현재 보유하고 교환 가능한지 다시 확인해 주세요.

  • 서로 줄 항목과 받을 항목을 한 문장에 모두 기록합니다.
  • 여러 장을 교환한다면 각 항목과 수량을 따로 나열합니다.
  • 조건이 바뀌면 기존 메시지를 수정했다고 가정하지 말고 새 조건을 다시 확인합니다.
  • 급하게 결정하도록 재촉받으면 목록과 게임 화면을 처음부터 다시 대조합니다.

4. 외부 링크와 개인정보

링크는 사이트가 보증하지 않습니다

교환글에 등록된 외부 연락처는 작성자가 입력한 값입니다. 운영자가 링크 목적지나 계정 소유권을 사전에 인증한 것이 아닙니다.

  • 로그인을 요구하는 낯선 페이지에서는 주소를 다시 확인합니다.
  • 파일 설치, 원격 제어, 화면 공유를 요구받으면 대화를 중단합니다.
  • 실명, 전화번호, 집 주소처럼 교환에 필요하지 않은 정보는 공개하지 않습니다.
  • 금전 거래나 계정 거래 제안은 스티커 목록 매칭이라는 사이트 목적을 벗어납니다.

5. 최종 교환 직전

  1. 1

    상대 확인

    대화한 닉네임과 실제 게임에서 마주한 닉네임이 같은지 확인합니다.

  2. 2

    항목 확인

    합의한 이름, 슬롯, 수량과 화면에 표시된 항목을 다시 비교합니다.

  3. 3

    변경 여부 확인

    최종 단계에서 항목이 바뀌지 않았는지 확인한 뒤 본인이 이해한 조건일 때만 진행합니다.

게임의 구체적인 교환 화면이나 기능은 업데이트로 달라질 수 있으므로, 이 사이트는 특정 버튼 순서나 피해 방지를 보장하지 않습니다.

6. 문제가 생겼을 때 남길 자료

  • 상대 닉네임과 문제가 발생한 날짜·시간
  • 합의한 스티커 이름, 슬롯, 수량이 보이는 대화
  • 해당 교환글이나 게시글을 식별할 수 있는 화면
  • 문제가 된 외부 링크가 있다면 주소가 보이는 화면
  • 개인정보와 인증정보를 가린 스크린샷

사이트의 신고 버튼으로 상황을 구체적으로 적거나 helper.dv3sticker@gmail.com으로 자료를 보낼 수 있습니다. 운영자는 접수 내용을 검토하지만 처리 시간이나 특정 제재 결과를 보장하지 않습니다.

7. 신고 기능의 범위

신고는 사이트 게시물과 이용 환경을 관리하기 위한 수단입니다. 게임 계정 제재, 손실된 아이템 복구, 법적 분쟁 해결을 대신하지 않습니다. 게임 안의 문제는 게임 운영사가 제공하는 공식 고객지원 절차도 함께 확인해야 합니다.
