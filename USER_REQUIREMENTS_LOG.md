본 문서는 사용자가 요청한 모든 기능 추가, 버그 수정, 그리고 디자인 의도 사항을 한곳에 누적하여 보관하는 통합 히스토리 대장입니다. 개발 진행 및 점검 시 항상 이 문서를 기반으로 정합성을 검증합니다.

## 📅 [2026-06-06] 피드백 및 요구사항 (v3.9 완료)

1. **🛠️ 내 글 수정 기능 (Edit Post)**
   - *의도/현상*: 내가 올린 교환 글을 지우고 다시 쓰는 번거로움을 해결하기 위해 본인 글 수정 기능을 요구함.
   - *해결 방안*: 피드 카드 및 내 정보 사이드 드로어 내에 '수정' 버튼을 추가하고, 수정 모달창을 개발했습니다. 수정 모달 내에 연락처 변경 기입란 뿐만 아니라 줄 수 있는 카드와 받고 싶은 카드를 모달 안에서 편하게 터치하여 실시간으로 가감(추가/제거)할 수 있는 직관적인 컴팩트 스티커 도감 바구니 셀렉터를 함께 설계하여 사용자 경험을 극대화했습니다.

2. **💬 댓글 기능 (Post Comments)**
   - *의도/현상*: 교환 글 피드 카드 내에서 다른 유저들과 자유롭게 피드백을 주고받거나 예약 상태 등을 논의할 수 있는 댓글 영역을 요구함.
   - *해결 방안*: 카드 하단에 '💬 댓글 (N)' 아코디언 버튼을 제공하여 토글이 가능하도록 만들었습니다. 댓글 목록 조회, 실시간 댓글 등록 및 댓글 작성자 본인/관리자 전용 삭제 권한 핸들러를 완벽 매핑했습니다.

3. **🚨 신고 기능 (Report / Flag)**
   - *의도/현상*: 허위 매칭 글, 사기 의심 연락처, 욕설 및 비방 댓글 등을 운영진에게 신고하여 정화할 수 있는 수단을 요구함.
   - *해결 방안*: 피드 카드 내에 빨간색 사이렌 신고 아이콘과 댓글 옆에 신고 버튼을 배치하고, 신고 사유(허위 정보, 사기 의심, 욕설/비방, 기타 수기 작성 등)를 선택 및 입력할 수 있는 신고 모달창을 추가하여 관리 시스템으로 인계되도록 구현했습니다.

4. **👑 관리자 전용 대시보드 구축 및 `간장` 계정 관리자 지정**
   - *의도/현상*: 닉네임 `간장` 로그인 시 사이트를 관리할 수 있는 전용 기능 및 신고 내역 모니터링 관리 도구를 요구함.
   - *해결 방안*: 로그인한 닉네임이 `간장`일 경우 헤더 닉네임 우측에 👑 관리자 배지가 추가로 표시되며, 클릭 시 전용 대시보드 모달이 오픈됩니다. 대시보드에서는 유저들이 신고한 실시간 누적 신고 목록을 모니터링하고, 원클릭으로 '신고 반려(무시)' 처리하거나 문제가 되는 게시글/댓글을 '강제 삭제 & 종결' 제재 조치할 수 있도록 강력한 중앙 관리 시스템을 완벽 구축했습니다.

## 📅 [2026-06-06] 이전 피드백 및 요구사항 (v3.8 완료)

1. **💬 실시간 대화방 정렬 및 실시간 업데이트 연동**
   - *의도/현상*: Supabase 데이터베이스 연동 이후 필드명 불일치로 인해 정렬, 날짜 표시, 알림이 정상적으로 작동하지 않음.
   - *해결 방안*: `chatService.js`에 중앙 포맷터를 탑재하여 snake_case를 camelCase로 완전히 매핑 완료하고, 메시지 발송 또는 수신 시 3초 타이머 대기 없이 즉각 대화방 목록을 리프레시 정렬하도록 고도화함.

2. **🕒 대화방 목록 업데이트 시간 표시 고도화**
   - *의도/현상*: 대화방의 마지막 업데이트 시각 표시를 직관적이고 미려하게 출력하도록 요구함.
   - *해결 방안*: `formatChatTime` 헬퍼 함수를 추가하여 오늘 메시지는 `오후 10:24`와 같은 형태, 어제 메시지는 `어제`, 7일 이내는 요일(예: `토요일`), 그 이전은 `6월 6일`과 같이 세련되게 분기 처리함.

3. **🚀 내 글 끌어올리기 (끌올) 기능 추가**
   - *의도/현상*: 내가 예전에 등록한 교환 등록글을 삭제하고 다시 쓸 필요 없이, 원터치 클릭 한 번으로 피드 목록의 맨 위로 올리는 기능을 요구함.
   - *해결 방안*: `bumpPost` API 서비스를 신설하여 Supabase `posts` 테이블의 `created_at` 필드를 현재 시각으로 갱신하고 피드를 다시 불러옴. 이 기능을 피드 카드 내 '내 글' 액션 영역 및 우측 '내 정보' 드로어 탭 목록에 동시 배치 완료함.
